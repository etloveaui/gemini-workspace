## ⚠️ **중요 주의사항**

### **절대 삭제 금지 파일들**
- `*.md` 파일 중 완성된 문서나 중요 계획서
- 프로젝트 관련 설정 파일
- 백업이 없는 유일한 파일들
- 크기가 큰 미디어 파일 (별도 확인 필요)

### **Git 관리 주의사항**  
- `projects/` 폴더 내용은 절대 건드리지 말 것
- 메인 워크스페이스 Git에만 영향을 주어야 함
- 아카이브 이동 시 Git 히스토리 보존 확인

### **에러 발생 시 대응**
- 파일 삭제 실패 → Python `os.remove()` 사용
- 권한 오류 발생 → 관리자 권한으로 재시도
- 중요 파일 실수 삭제 → 즉시 백업에서 복구
