# Gemini CLI 도움말

## 시작하기

이 섹션에서는 Gemini CLI를 시작하는 방법에 대한 정보를 제공합니다.

## 명령어 개요

사용 가능한 명령어에 대한 개요입니다:

* `invoke doctor`: 환경 설정을 확인합니다.
* `invoke quickstart`: 빠른 시작 가이드를 확인합니다.
* `invoke help [section]`: 도움말 정보를 표시합니다.
* `invoke search "<query>"`: 웹 검색을 수행하고 결과를 요약합니다.
